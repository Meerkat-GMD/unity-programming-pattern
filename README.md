# Unity Programming Pattern

경기게임마이스터고 방과후학교 Unity 프로그래밍 패턴 수업 자료 저장소입니다.

각 차시는 독립 폴더로 관리합니다.

```text
02차시_CommandPattern/
  강의자료/
    유니티 프로그래밍 패턴 2차시 - Command Pattern.pptx
  실습파일/
    CommandPatternUnityPractice/
  미리보기/
    PPT 렌더링 검수 이미지

03차시_Flyweight/
  강의자료/
    유니티 프로그래밍 패턴 3차시 - Flyweight.pptx
  실습파일/
    FlyweightUnityPractice/
  미리보기/
    PPT 렌더링 검수 이미지
```

## 수업 자료 사용 방법

1. 원하는 차시 폴더를 엽니다.
2. `강의자료`에서 PPT를 확인합니다.
3. `실습파일`의 Unity 프로젝트를 Unity Hub에서 엽니다.
4. `Assets/Scenes` 안의 데모 씬을 열고 Play 버튼을 눌러 결과를 확인합니다.

## 현재 차시

- `02차시_CommandPattern`: Command Pattern 강의 PPT와 Unity 실습 프로젝트
- `03차시_Flyweight`: Flyweight Pattern 강의 PPT와 Unity 실습 프로젝트

## 관리 규칙

- 새 차시는 `NN차시_PatternName` 형식의 폴더로 추가합니다.
- PPT는 각 차시의 `강의자료` 폴더에 둡니다.
- Unity 실습 프로젝트는 각 차시의 `실습파일` 폴더에 둡니다.
- Unity의 `Library`, `Temp`, `Obj`, `Build`, `Logs`, `UserSettings` 폴더는 Git에 올리지 않습니다.
