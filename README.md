# Unity Programming Pattern

Unity 프로그래밍 패턴 방과후학교 수업 자료 저장소입니다.

각 차시는 폴더 단위로 관리합니다.

```text
02차시_CommandPattern/
  강의자료/
    유니티 프로그래밍 패턴 2차시 - Command Pattern.pptx
  실습파일/
    CommandPatternUnityPractice/
```

## 운영 규칙

- 차시별 폴더명은 `NN차시_패턴명` 형식으로 만든다.
- PPT는 `강의자료` 폴더에 넣는다.
- Unity 실습 프로젝트와 스타터 코드는 `실습파일` 폴더에 넣는다.
- Unity가 자동 생성하는 `Library`, `Temp`, `Obj`, `Build`, `Logs` 등은 git에 올리지 않는다.

## 현재 포함된 자료

- `02차시_CommandPattern`: Command Pattern 강의 PPT 및 Unity 실습 프로젝트
